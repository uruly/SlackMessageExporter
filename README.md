## Usage

CSV
```
deno run --env --allow-net --allow-write --allow-read --allow-env save_csv.ts
```

HTML
```
deno run --env --allow-net --allow-write --allow-read --allow-env save_html.ts
```

## Permissions

- `channels:history`
- `emoji:read`
- `files:read`
- `users:read`
